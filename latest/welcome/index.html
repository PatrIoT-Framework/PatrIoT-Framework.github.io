PatrIoT Framework Documentation

Welcome to the Patriot Framework Documentation.

The PatrIoT framework enables testing of distributed IoT applications by providing test developers with features such as network virtualization, data simulation, and/or interaction with hardware devices all integrated into a single tool.

Getting started guide

The first place which will help you to orient yourself with the framework is the Getting started guide which describes the first steps to understand and setup necessary tools and environment to enable efficient development of tests.

As part of getting started you can familiarize yourself with example project which shows full suite of integration tests that uses PatrIoT Framework to facilitate specifics related to IoT, if you are interested navigate yourself to the Smart home tests.
